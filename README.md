# Aastrbot_plugin_animewifex

原插件：https://github.com/zgojin/astrbot_plugin_AW

在此基础上添加了几个功能，更改了数据目录，支持插件面板配置。

**本插件代码为AI生成，下面图床的也是。**

配套图床：https://github.com/monbed/wife

也可以手动下载图片，放入AstrBot\data\plugin_data\astrbot_plugin_animewifex\img\wife目录。

## 指令 ##
- `抽老婆` 每天一次，随机抽一张二次元老婆
- `查老婆` 查看今日老婆 加@可以查看别人老婆
- `切换ntr开关状态` 现在默认开启
- `牛老婆` @用户概率牛别人老婆
- `换老婆` 重新抽取老婆
- `重置牛` 重置牛老婆次数，也可@用户重置别人的次数，失败禁言300秒。增加AstrBot管理员权限，不受限制。
- `重置换` 重置换老婆次数，其余同上，与重置牛共享次数。合并配置。
- `交换老婆` @用户和对方交换老婆 插件可配置每日可使用次数
- `同意交换` @用户同意
- `拒绝交换` @用户拒绝
- `查看交换请求` 查看交换老婆请求
- `发老婆` 【管理员】@用户 老婆名 → 派发受保护的老婆（无法被牛/交换）
- `拆散` 【管理员】@用户 → 清空所有老婆；@用户 老婆名 → 只拆散指定的那个老婆
- `解除保护` 【管理员】@用户 → 解除保护状态（含纯爱模式）但不拆散老婆，使其可以被牛/交换

## 💕 纯爱模式 ##

当抽到的老婆来自指定出处（默认「尘白禁区」）时，自动激活**纯爱模式**：

### 效果
- 🛡️ **纯爱守护**：纯爱老婆自动获得保护，无法被牛、无法交换、无法更换
- 📅 **跨天坚持**：纯爱老婆不会在每日刷新时消失，每天抽老婆时天数+1
- 🔄 **主格子轮换**：每服务满 N 天（默认3天），主格子尘白老婆自动随机轮换为新的尘白角色（排除当前角色），并显示轮换提示
- 🎁 **坚持奖励（副格子）**：每坚持 N 天（默认7天），获得1次额外抽老婆机会，副格子最多 3 个
- 🔄 **副格子已满时替换**：当副格子满3个且触发新奖励时，提供 A/B/C/D 选项（A=主格子，BCD=副格子），回复字母进行替换，回复"不换"或24点前未回复则保留现状
- ✨ **特殊尘白副格子**：奖励抽到尘白角色时，不占副格子名额，放入独立的特殊格子（默认上限1个），无法轮换、无法被替换
- ⚠️ **替换主格子解除纯爱**：如果选择替换A（主格子），纯爱模式将被解除
- 🚫 **角色不重复**：纯爱格子（主格子+副格子+特殊格子）之间角色互不重复
- ⚠️ **纯爱考验**：纯爱用户如果去牛别人的老婆，将**暗中100%失败**，且有一定概率导致纯爱老婆**跑路**
- 💔 **跑路惩罚**：纯爱老婆跑路后，纯爱模式解除、**所有老婆一起跑路**（含副格子和特殊格子）、当天无法再抽老婆（只能去牛别人的）
- 📊 **超标清理**：如果历史数据中副格子超过上限，自动移除多余的，并补偿减少等待时间（每移除1个补偿3天）

### 配置项
| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `pure_love_enabled` | bool | true | 是否启用纯爱模式 |
| `pure_love_source` | string | 尘白禁区 | 触发纯爱模式的角色卡出处关键词 |
| `pure_love_runaway_prob` | float | 0.30 | 纯爱用户牛别人时，纯爱老婆跑路的概率 |
| `pure_love_reward_days` | int | 7 | 每坚持多少天获得1次额外抽老婆机会（副格子） |
| `pure_love_rotation_days` | int | 3 | 主格子尘白老婆每服务多少天后自动轮换 |
| `pure_love_max_bonus` | int | 3 | 纯爱副格子最多可拥有的数量 |
| `pure_love_special_slots` | int | 1 | 特殊尘白副格子的数量上限 |
| `pure_love_blacklist` | string | （空） | 奖励抽取时排除的角色关键词，英文逗号分隔 |

### 触发条件
角色卡文件名格式为 `出处!角色名.扩展名`（如 `尘白禁区!琳恩.png`），当出处部分包含配置的关键词时自动激活。
